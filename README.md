# Tecnicatura en Programacion 1° Semestre

En este recorrido que hicimos por el primer semestre de la ***Tecnicatura en programacion de la Universidad Tecnológica de San Rafael***, nos adentramos en los lenguajes de programación favoritos por todos los desarrolladores, tales como **Python, JavaScript y Java.**

**Python:** Con este lenguaje empezamos a conocerlo y a hacer ejercicios dentro de entorrnos virtuales. Se pueden ver los trabajos realizados en el primer semestre [aqui](https://github.com/juanmarcoso/Tecnicatura/tree/main/Python)

**JavaScript:** Con este lenguaje, vimos los principios de HTML y las funciones básicas de JavaScript. Sus similitudes con Java y sus diferencias. Sus formas de poder visualizarlo, tanto con Quokka como desde una web generando un archivo index.html. Si quieres ver mis trabajos del primer semestre pincha [aqui](https://github.com/juanmarcoso/Tecnicatura/tree/main/Javascript).

**Java:** Este lenguaje tan popular, se comenzó a trabajar en la tecnicatura. Aprendimos muchos recursos para escribir codigo rápido y entender la sintaxis. Si quiere ver más trabajos, pincha [aqui](https://github.com/juanmarcoso/Tecnicatura/tree/main/Java).

Tambien empezamos a hacer nuestro primer **Portafolio Web** con **HTML** y **CSS**. Usaos distitnas herramientas que se veran reflejados en el siguiente link del [Portafolio](https://github.com/juanmarcoso/Portfolio)

Por si fuera poco, tambien pudimos trabajar muchisimos ejercicios en **PeSeInt** un lenguaje en pseudocodigo ideal para el aprendizaje de la lógica de programación. Esta carpeta puedes verla [aqui](https://github.com/juanmarcoso/PseIntTrabajos).
