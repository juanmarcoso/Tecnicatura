-=

/=

**= (exponente igual)

Operadores aritmeticos:
+
-

Operadores de asignacion:
=

Operadores de comparacion (Devuelve un tipo de valor booleano):
== Revisa el valor sin importar el tipo de dato
=== Revisa el valor y el tipo tambien (Igualdad estricta)

Documentar
// = Para una linea de comentario
/**/ = Para multiples lineas de comentario

Secuencias de escape:
\' = Mostrar una comilla simple dentro de una cadena en la salida de informacion
\" = Mostrar doble comilla dentro de una cadena en la salida de informacion
\\ = Barra invertida, diagonal inversa, es para mostrar una back slash
\n = Salto de linea
\r = Retorno
\t = Tabulacion
\b = Retroceso
\f = Salto de pagina (para impresoras)

Operador de aumento:
++
Ejempĺo: Let z = ++a; //PreIncremento, en z le va a venir el valor de una variable pero sumandole un numero mas
Ejemplo 2: z = a++; //PosIncremento, esto significa que va a recibir el valor de la variable 'a', que quiere decir, que cuando utilicemos z, va a valer el valor a, pero cuando volvamos a utilizarla, ya va a valer un numero mas.

Operador de decremento:
--
Ejempĺo: Let z = --a; //PreDecremento
Ejempĺo: Let z = a--; //PosDecremento

Operador Distinto o Diferente (Devuelve un tipo de valor booleano):
!= //Comprueba valores sin revisar e tipo
!== //Comprueba el valor y tambien el tipo

Operadores Relacionales (Devuelve un tipo de valor booleano):
<
>
>=
<=

Palabras reservadas:
